## What It Does

Glama indexes the Model Context Protocol ecosystem — submit a GitHub repo and it indexes
every tool, schema, and annotation. Servers appear in search, categories, and
recommendations. An in-browser inspector lets visitors exercise tools with real JSON-RPC
calls without cloning or building anything locally.

## Key Features

- **Registry & discovery**: 50,000+ servers indexed, filterable by category, claimed
  status, and server type; quality-scored with verification tiers (Official, Claimed,
  crawled)
- **In-browser inspector**: test any listed server's tools with live JSON-RPC calls
- **Gateway**: request logging, per-tool access control, managed OAuth, usage analytics
  — sits in front of MCP servers as a proxy
- **Connectors**: pre-configured hosted MCP servers with managed credentials; paste the
  Glama URL into Claude, ChatGPT, or Cursor and it works without local server setup

## Verification Tiers

- **Official**: publisher-verified entries
- **Claimed**: author has verified ownership
- **Crawled**: automatically indexed, unverified

## Comparable Registries

Other MCP registries include mcp.so (~20k servers), Smithery.ai, and the
punkpeye/awesome-mcp-servers GitHub list.
