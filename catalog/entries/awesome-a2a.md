## What it does

Curated resource collection for the Agent2Agent (A2A) protocol ecosystem. Covers:

- **Official resources**: Spec, documentation, Google's samples
- **SDKs by language**: Python (`a2a-sdk`), Go, JS (`@a2a-js/sdk`), Java, .NET, Rust
- **Framework integrations**: Official samples for Google ADK, LangGraph, CrewAI, Genkit; community implementations spanning dozens of frameworks
- **Platforms & integrated solutions**: Commercial and open-source platforms with A2A support
- **Tools & utilities**: Agent Cards, testing, observability, gateways
- **Tutorials & articles**: Getting-started guides, architecture deep-dives, comparisons with MCP

Organized by category with brief descriptions and links. Actively maintained as the A2A ecosystem grows.

## Assessment

Useful as a discovery index for the A2A ecosystem — same role as "Awesome Claude Code" plays for Claude Code tooling. The A2A protocol is worth tracking for multi-agent interoperability scenarios where agents from different frameworks (or different organizations) need to collaborate without sharing internal state.

For this catalog, the primary value is as a source of future inbox items — specific A2A implementations or tools worth evaluating individually. The protocol itself is cataloged separately in the a2a-protocol entry.
