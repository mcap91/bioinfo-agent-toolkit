## What it does

ChatBotKit is a commercial AI agent platform providing operational control for deploying and running AI agents. Features include enterprise security (content filtering, data protection, GDPR/CCPA compliance), built-in OAuth and MCP (Model Context Protocol) integration, observability dashboards for monitoring conversations and debugging, and policy management for content policies, rate limits, and access controls.

Architecture is modular — agents are composed from reusable datasets, skillsets, and integrations.

## Differentiators

- **MCP-native**: Built-in Model Context Protocol support for connecting agents to external services without custom auth code
- **Forward Deployment platform**: Enables rapid prototyping and customer validation of AI products — case study (Quench/enterprise search) reports 10x faster prototyping with Fortune 500 clients including Yum Brands, MotorK, Podium
- **Data governance**: Bring-your-own-model, regional data residency, full data ownership controls
- **Cross-domain case studies**: Enterprise search (Quench), healthcare (DEBRA UK — rare genetic conditions), K-12 education wellbeing (Elggo — 30% wellbeing increase across 12,000 students in 32 schools), cultural heritage (FARO — museum guide with geofencing), multilingual customer service (Intelliway — 40% positive feedback increase)

## Mechanical details

Commercial SaaS platform with API and SDK access. Embeddable widget for web deployment. Supports multilingual content delivery. No self-hosted option described in public documentation.

## Security

Proprietary commercial platform. Claims built-in content filtering, data protection, and compliance for GDPR, CCPA, and industry-specific requirements. OAuth integration for third-party service connections. No open-source code to audit.
